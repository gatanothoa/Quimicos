# 🐕 Tienda de Limpieza del Chamaco Perro

Demos profesionales para empresa de productos químicos de limpieza industrial y comercial.

## 📋 Descripción

Este repositorio contiene dos versiones demo de un sitio web comercial para "Tienda de Limpieza del Chamaco Perro", una distribuidora especializada en productos químicos de limpieza de alta calidad.

## 🚀 Versiones Disponibles

### 📱 **Versión Simple** (`/Simple/`)
- **Archivo único**: `index.html` con CSS y JavaScript embebidos
- **Deploy inmediato**: Listo para subir a cualquier hosting
- **Funcionalidades**:
  - Catálogo de 6 productos químicos
  - Carrito de compras funcional
  - Formulario de contacto
  - Integración WhatsApp
  - Diseño responsive

### 💼 **Versión Compleja** (`/Compleja/`)
- **Estructura profesional**: Archivos separados por tipo
- **Arquitectura modular**: JavaScript organizado en módulos
- **Funcionalidades avanzadas**:
  - Catálogo de 10 productos químicos completos
  - Sistema de búsqueda y filtros
  - Animaciones sofisticadas
  - Gestión de estado avanzada
  - Manejo robusto de errores
  - Performance optimizado

## 🛠️ Tecnologías Utilizadas

- **HTML5** semántico y accesible
- **CSS3** con variables custom y grid/flexbox
- **JavaScript ES6+** vanilla (sin frameworks)
- **Font Awesome 6.4.0** para iconografía
- **Google Fonts** (Poppins) para tipografía
- **LocalStorage** para persistencia de datos

## 🎨 Características de Diseño

- **Glassmorphism effects** modernos
- **Gradientes dinámicos** y atractivos
- **Animaciones suaves** optimizadas para GPU
- **Responsive design** mobile-first
- **Accesibilidad completa** (WCAG 2.1)
- **Modo oscuro** considerado
